# Explain Simply

Use this when an explanation feels too abstract or technical.

## Prompt

Explain `[concept]` in plain language for `[audience]`.

Context: `[where I encountered it or why it matters]`

Use one concrete example and, if helpful, one analogy. Define unavoidable
technical terms when they first appear. Explain what it is, how it works, why it
matters, and where the simplified explanation stops being accurate.

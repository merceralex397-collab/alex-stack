# Diagnose a Problem

Use this to investigate causes before choosing a fix.

## Prompt

Help me diagnose this problem:

`[problem description]`

Expected result: `[what should happen]`

Actual result: `[what happens]`

When it started and what changed: `[details]`

Evidence already available: `[observations, errors, data, or attempted fixes]`

Constraints: `[limits or safety concerns]`

Separate observations from hypotheses. Rank likely causes, identify the evidence
for and against each one, and propose the safest, most informative next checks.
Do not jump to a solution before establishing the likely cause.
